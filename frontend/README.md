# Attendance Mobile

A minimal React Native TypeScript app for student attendance scanning and check-in.

## Screens

- NFC scanner with animated signal visualization.
- Student detail screen that displays backend-provided data.
- Add student screen for storing card UID, full name, index number, and reference.
- Missing check-in screen with manual index-based check-in.

## Run

1. Install dependencies:

   ```bash
   cd codequest/
   npm install
   ```

2. Start Expo:

   ```bash
   npm start
   ```
